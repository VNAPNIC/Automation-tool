Hướng dẫn test

  1. Mở extension Element Selector
  2. Click "Bắt đầu chọn element"
  3. Hover qua các element trên trang này
  4. Click vào element muốn lấy selector
  5. Kiểm tra selector đã được copy vào clipboard
  6. Paste selector vào console hoặc editor để test
